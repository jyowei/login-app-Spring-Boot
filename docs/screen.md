# 画面遷移図

## ログイン

[ログイン画面] (/login)

    ├─ 成功（POST /login）
    
    │   └→ [マイページ] (/mypage)
    └─ 失敗（POST /login）
        └→ [ログイン画面（エラー表示）]

## 新規登録

[登録画面] (/signup)
    ├─ 成功（POST /signup）
    │   └→ [ログイン画面]
    └─ 失敗（POST /signup）
        └→ [登録画面（エラー表示）]

## マイページ

[マイページ] (/mypage)
    ├─ 未ログイン
    │   └→ [ログイン画面]
    └─ ログアウト（GET /logout）
        └→ [ログイン画面]
