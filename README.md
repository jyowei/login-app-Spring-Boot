# Login Firebase App


## デモ動画

![ログイン画面](demo.gif)

## 概要
Spring Boot と Firebase Authentication を使用したログイン機能付きWebアプリです。

## 機能
- ユーザー登録（Firebase）
- ログイン機能
- エラーメッセージ表示
- Thymeleafによる画面制御

## 使用技術
- Java 17
- Spring Boot
- Thymeleaf
- Firebase Authentication
- Gradle

http://localhost:8080/login
